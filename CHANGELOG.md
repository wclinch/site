# Changelog

## 1.0.0-beta.1 — 2026-05-14

First public beta.

### What's included
- Stack — left column. Sources (uploaded files) on top, Sites (saved pages) on bottom.
- Source — center column. Two independent panels, Source 1 and Source 2. Each opens a PDF or image from the Stack.
- Research — right column. Live browser with URL bar. Save any page to Sites with one click.
- All data stays on the device. No account, no server, no telemetry.
- 250 MB cap for uploaded Sources. Saved Sites store metadata only and do not count toward storage.
- State restores on relaunch: Stack, open Sources, and Research URL.
- License key activation via Polar (one-time, validates once, app works offline after that).

### Platform
- macOS arm64 (Apple Silicon)
- macOS x64 (Intel)
